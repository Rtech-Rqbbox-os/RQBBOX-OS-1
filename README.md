## What is RQBBOX OS 1?

RQBBOX OS 1 is the **umbrella application** for the entire RQBBOX ecosystem. It's a hub that connects all RQBBOX apps — gaming, desktop, simulation — into one unified experience.

---

## The Ecosystem

| App | Description | Type |
|-----|-------------|------|
| **[RQBBOX MODE](https://rtech-rqbbox-os.github.io/RQBBOX-MODE/)** | Xbox-style gaming dashboard with 7 built-in games | Electron .exe |
| **[RBOWS 1](https://rtech-rqbbox-os.github.io/RBOWS-1/)** | Windows 10 desktop environment with 10+ built-in apps | Electron .exe |
| **[RQBBOX LIFE](https://rtech-rqbbox-os.github.io/rqbbox-life/)** | Neon life simulation game | Electron .exe |
| **RQBBOX OS 1** | The hub that ties everything together | Electron .exe |
